# Proyecto Frontend — Gestión de Usuarios y Carreras de Taxi

Este proyecto es una aplicación frontend desarrollada con **React + TypeScript + Vite** que permite gestionar usuarios y carreras de taxi consumiendo las APIs de un backend en **Spring Boot** (arquitectura DDD/hexagonal, con persistencia en memoria).

## Ejecutar el servidor de desarrollo

Desde la carpeta raíz del proyecto (`final-webdev`), ejecuta:

```bash
npm install
npm run dev
```

Luego abre en el navegador la URL que indique Vite (por defecto: `http://localhost:5173`).
